## html中的标签分类
- 行内
    - 水平方向都在一行上面
    - 垂直方向：以当前行的base-line 对齐
    - 不能设置宽高(是错误的说法)
    - 非置换元素不能设置宽高
    - 行内置换元素是能设置宽高的
- 块级
    - 独占一行
    - 能设置宽高
    - (BFC):块级格式上下文
- 置换元素：
    - 内部内容不受css控制
    ```html
    <img src="">
    <canvas />
    <input />
    <iframe src="">
    <script src="">
    ```
- 非置换元素：
    ```html
    <div>
    ```
- 词法(静态)作用域：js c c++
- 动态作用域 bash
