# 1588. 所有奇数长度子数组的和
    给你一个正整数数组 arr ，请你计算所有可能的奇数长度子数组的和。
    子数组 定义为原数组中的一个连续子序列。
    请你返回 arr 中 所有奇数长度子数组的和 。
    示例 1：
    输入：arr = [1,4,2,5,3]
    输出：58
    解释：所有奇数长度子数组和它们的和为：
    [1] = 1,[4] = 4,[2] = 2,[5] = 5,[3] = 3,[1,4,2] = 7,[4,2,5] = 11,[2,5,3] = 10,[1,4,2,5,3] = 15,
    我们将所有值求和得到 1 + 4 + 2 + 5 + 3 + 7 + 11 + 10 + 15 = 58
- 解题思路
    - 嵌套三层循环，第一层循环有几种奇数个数组(1,3,5,7...)
    - 第二层遍历数组中的每一项
    - 第三层按规则把子数组的项加起来
